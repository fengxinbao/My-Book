cat

只显示包含某些字符的行

cat filename | grep "regex"

例如，显示 file_list 中以 maven 开头的行

cat file_list | grep "^maven"

tree

以树形结构显示目录结构

$ apt-get install tree
$ tree -L 5 src/

在命令行中创建和使用一个变量

$ p=`pwd`
$ cd
$ mv Gradle $p

安装 Gradle

解压 gradle-4.7-bin.zip

sudo unzip -d /opt/gradle/ gradle-4.7-bin.zip

将 Gradle 的 bin 目录添加至环境变量

$ export PATH=$PATH:/opt/gradle/gradle-4.7/bin

查看 Gradle 是否安装成功

$ gradle -v

列编辑

删除

  1. 光标定位到开始操作的地方
  2. Ctrl + v 进入“可视块”模式
  3. 选取要操作的列
  4. 用 d 删除

插入: 假设我们在每一行前都插入#：

  1. 光标定位到开始操作的地方
  2. Ctrl + v 进入“可视块”模式
  3. 选取要操作的列
  4. Shift + I
  5. 输入要插入的内容。
  6. <ESC> 按两次，会在每行的选定区域出现插入的内容
