Linux Tips

Run a program in the background

firefox deom.html &

Set the Environment Variables

Append this line

export PATH=/path/to/the/bin/directory:$PATH

to the end of the file ~/.bashrc

在命令行中创建和使用一个变量


$ p=`pwd`
$ cd
$ mv Gradle $p

安装 Gradle

解压 gradle-4.7-bin.zip

sudo unzip -d /opt/gradle/ gradle-4.7-bin.zip

将 Gradle 的 bin 目录添加至环境变量

$ export PATH=$PATH:/opt/gradle/gradle-4.7/bin

查看 Gradle 是否安装成功

$ gradle -v
