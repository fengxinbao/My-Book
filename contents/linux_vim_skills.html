Vim Skills

Exchange two letters in normal mode

xp

Exchange two lines in normal mode

ddp

Insert an empty line in normal mode

  1. Move the cursor to an empty line, yy.
  2. Move the cursor to the target postion, p.

INSERT the start/end of the line

I / A

列编辑

删除

  1. 光标定位到开始操作的地方
  2. Ctrl + v 进入“可视块”模式
  3. 选取要操作的列
  4. 用 d 删除

插入: 假设我们在每一行前都插入#：

  1. 光标定位到开始操作的地方
  2. Ctrl + v 进入“可视块”模式
  3. 选取要操作的列
  4. Shift + I
  5. 输入要插入的内容。
  6. <ESC> 按两次，会在每行的选定区域出现插入的内容
